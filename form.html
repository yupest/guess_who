Создай свою мультивселенную! Выбери персонажей из предложенных вселенных для создания своей коллекции. Перейди в мир вселенной по клику из левого списка, чтобы открыть карточки персонажей. Кликай на карточки персонажей, чтобы добавить в коллекцию. Двойной клик по названию мира добавит в коллекцию всех доступных персонажей. Чтобы удалить персонажа — дважды кликни по нему в коллекции. Максимальный размер коллекции — 48 персонажей.
